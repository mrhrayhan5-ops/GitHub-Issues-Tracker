1. Difference between var,let and const:

"var" is function-scoped and can be redeclared and reassigned, while "let" is block-scoped and can be reassigned but not redeclared, and "const" is block-scoped but cannot be reassigned or redeclared.

2. Spread operator(...):

 The spread operator ... is used to expand elements of an array or object.

3. Difference between map(),filter(), and forEach();

 map() creates a new arry by modifying every element, 
    filter() creats a new arry with elements that pass a condition,
    and forEach() only loops through elements without returning a new array.

4. Arrow function:

An arrow function is a shorter syntax for writing functions in JavaScript using the => symbol.

5. Template literals:

Template literals are strings written with backticks (`) that allow embedding variables and expressions using ${}.
